PLANMYPEAK IMPORTER - QUICK INSTALL
==================================

For the current installation guide, see:
- INSTALL.md
- README.md

Quick steps:
1. Build the extension or download a packaged ZIP.
2. Open chrome://extensions
3. Turn on Developer mode
4. Click Load unpacked
5. Select the dist/ folder or the extracted release folder
6. Sign in to app.trainingpeaks.com and refresh the page
7. Open the extension popup

Notes:
- Unpacked extensions do not auto-update
- Do not delete the unpacked folder after installing it
- For privacy and permissions, see PRIVACY.md
